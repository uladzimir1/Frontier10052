# Frontier 10052

AAA hard-science-fiction trading, exploration, crew, and civilization game set in the year 10052.

## Current implementation slice

- Cinematic marketing landing page
- Animated starfield and orbital traffic
- Game launcher/start route
- Responsive Apple-inspired interface
- .NET 10 Blazor Web App foundation

## Run locally

```bash
dotnet run --project src/Frontier10052.Web
```

Then open the HTTPS URL printed by ASP.NET Core.
